* Dependency:
	* Not any if anaconda3 with python3+ is installed
	* If No anaconda:
		* Install python3 plus
		* Install flask using "pip install flask"


* Move to directory "project xyz"
   you will see structure like this:
	* static
	* templates
	* readme.txt
	* speak.py


* Run file from terminal:
	python speak.py flask run


===============================================================
Few details you should know before you start working on project
===============================================================

* "static" folder must contains only js/css files

* "templates" folder must contain your html

* All your .py files should be in main project directory
